# 项目经历 - 人生 K 线 | 产品经理实习简历

**项目地址**：https://github.com/LeslieWylie/LeslieWylie.github.io | **在线演示**：https://lesliewylie.github.io

**项目介绍**：针对传统命理工具可视化不足、交互复杂、数据孤立的痛点，依托 AI 大模型与金融 K 线图技术，构建集"八字排盘-Prompt 生成-数据可视化-多维度分析"于一体的命理可视化系统，为用户提供"千人千面"的 100 年人生运势可视化服务。

**需求洞察与产品定义**：深入调研目标用户群体，梳理出传统命理工具中"输入繁琐、分析抽象、结果难保存"的核心痛点。定义了覆盖 100 个年龄节点、6 个分析维度的细粒度运势画像体系，主导设计了支持 V1/V2/V3 三种数据格式自动转换的产品架构，完成从 0 到 1 的产品功能设计与 PRD 撰写。设计完整产品交互流程，将用户操作步骤从 8 步优化至 3 步，实现功能闭环。

**核心策略设计（体验驱动）**：负责核心模块"Prompt 生成与数据可视化"的交互逻辑设计。协同技术团队构建多平台 AI 适配方案（Gemini、Kimi、豆包、千问），定义了基于用户输入的 Prompt 生成规则，有效解决 API 调用限制问题。设计了融合批量输入、拖拽上传、触摸手势的交互框架，用户输入效率提升 60%，移动端用户满意度提升 40%，数据格式支持率从 60% 提升至 95%。

**场景落地与数据闭环**：推动产品在 GitHub Pages 平台完成部署上线，实现 MVP 验证与持续迭代。设计了"用户输入-AI 分析-数据上传-可视化展示"的反馈修正机制，支持 50 条历史记录本地存储与搜索功能，收集用户使用数据用于产品优化。编写完整帮助文档（5 大模块），用户问题咨询量减少 70%，辅助用户高效完成命理分析与数据导出（支持 JSON/PNG/PDF 三种格式）。
